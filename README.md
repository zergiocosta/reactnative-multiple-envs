# React Native - Multiple Environments

#### POC for setting multiple environments in React Native

Setting up your environment:

```sh
  # installing dependencies
  $ yarn
```

Running on iOS:

```sh
  # local env
  $ yarn ios

  # homolog env
  $ yarn ios:homolog

  # production env
  $ yarn ios:production
```

Running on Android:

```sh
  # local env
  $ yarn android

  # homolog env
  $ yarn android:homolog

  # production env
  $ yarn android:production
```
